### 1. AI Tutor ü§ñ

**What is it?**
An intelligent chatbot that helps you learn by answering questions and providing explanations.

**How to use:**
1. Click **AI Tutor** from dashboard
2. Type your question in the chat box
3. Press send or hit Enter
4. Receive instant AI-powered responses

**Tips:**
- Ask specific questions for better answers
- Use it for homework help
- Request explanations of complex topics
- Ask for examples and practice problems

**Example Questions:**
- "Explain photosynthesis"
- "How do I solve quadratic equations?"
- "What is the Pythagorean theorem?"

**Class Status:**
- üü° **Scheduled**: Upcoming class

## Troubleshooting

### Common Issues

**Cannot Login:**
- Check email and password
- Verify internet connection
- Clear app cache
- Contact admin if issue persists

**Video Not Working:**
- Check camera permissions
- Test camera in device settings
- Restart app
- Try different browser (web version)

**Audio Issues:**
- Check microphone permissions
- Test microphone in settings
- Check volume levels
- Use headphones

**Class Not Loading:**
- Refresh the page
- Check internet connection
- Clear browser cache
- Try different browser

**Translation Not Working:**
- Check internet connection
- Try offline mode
- Restart app
- Update app to latest version

## FAQs

### General Questions

**Q: Is JeduAI free to use?**
A: Contact your institution for pricing and licensing information.

**Q: What devices are supported?**
A: Android, iOS, Windows, Mac, and web browsers.

**Q: Can I use JeduAI offline?**
A: Limited features work offline, including basic translations.

### Student Questions

**Q: How do I join a class?**
A: Click the "Join Class" button when it appears (10 minutes before start).

**Q: Can I review past classes?**
A: Yes, recordings are available in the Online Classes section.

**Q: How do I submit an assessment?**
A: Complete all questions and click the "Submit" button.

**Q: Can I retake an assessment?**
A: Depends on teacher settings. Contact your teacher.

### Staff Questions

**Q: How many students can join a class?**
A: Default is 50, but can be configured up to 100.

**Q: Can I edit an assessment after publishing?**
A: Yes, but student submissions may be affected.

**Q: How do I grade assessments?**
A: Go to Assessment Management and select the assessment to grade.

### Admin Questions

**Q: How do I add bulk users?**
A: Use the CSV import feature in User Management.

**Q: Can I customize the platform?**
A: Yes, through System Settings and branding options.

**Q: How is data backed up?**
A: Automatic daily backups to Supabase cloud storage.
