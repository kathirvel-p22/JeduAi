HOT RELOAD (While Running):
----------------------------
In the terminal where the app is running:
- Press 'r' → Hot reload (fast refresh)
- Press 'R' → Hot restart (full restart)
- Press 'q' → Quit the app

RESPONSIVE DESIGN:
------------------
Test on different screen sizes:
1. Press F12 in Chrome (Developer Tools)
2. Click the device toolbar icon (phone/tablet icon)
3. Select different devices (iPhone, iPad, etc.)
4. See how the app adapts!

STOP THE APP:
-------------
To stop the web server:
1. Go to the terminal
2. Press 'q' or Ctrl+C

RESTART THE APP:
----------------
cd JeduAi
flutter run -d chrome --web-port=8080

TROUBLESHOOTING:
----------------
Issue: Port 8080 already in use
Solution: Use different port:
flutter run -d chrome --web-port=8081

Issue: Chrome doesn't open
Solution: Manually open http://localhost:8080

Issue: Blank page
Solution: Check browser console (F12) for errors

Issue: Features not working
Solution: Make sure all dependencies are installed:
flutter pub get
